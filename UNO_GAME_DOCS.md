# UNO 纸牌游戏 - HTML 界面文档

## 项目简介

这是一个基于浏览器的 UNO 纸牌游戏，使用纯 HTML + CSS + JavaScript 实现，无需任何外部依赖。支持 1 名玩家对战 3 名 AI，包含完整的 UNO 规则、真实卡牌图片、音效系统和智能 AI。

---

## 功能特性

### 🎮 游戏玩法
- **1v3 对战**：玩家（下方）vs AI一号（左侧）、AI二号（上方）、AI三号（右侧）
- **完整 UNO 规则**：颜色匹配、数字匹配、功能牌、万能牌
- **罚牌叠加机制**：+2 和 +4 可以叠加，直到有人无法叠加而抽牌
- **UNO 喊叫**：手牌剩 1 张时 AI 会自动喊 UNO

### 🎨 界面设计
- **四方位布局**：玩家坐下方，三名 AI 分别坐左、上、右
- **真实卡牌图片**：使用 `Cards/` 文件夹中的 UNO 官方风格卡牌图片
- **方向指示器**：右上角显示当前出牌方向（顺时针/逆时针）
- **当前玩家高亮**：轮到谁出牌时，该玩家名称会金色/绿色高亮并闪烁
- **手牌悬停效果**：鼠标悬停时卡牌放大并上浮

### 🔊 音效系统
- 使用 Web Audio API 生成音效，无需外部音频文件
- 包含：出牌、抽牌、跳过、反转、罚牌、胜利、点击、错误等 8 种音效
- 左上角有音效开关按钮

### 🤖 AI 智能
- 基于评分系统的策略选择
- 优先出高数字牌和功能牌
- 万能牌根据手牌最多的颜色智能选择
- 保留万能牌到最后使用
- 20% 随机性避免过于可预测

## 使用说明

### 启动游戏
直接用浏览器打开 `uno-game.html` 即可，无需服务器。

### 玩家操作
1. **出牌**：点击手牌中高亮（可出）的卡牌
2. **抽牌**：点击中央的抽牌堆
3. **选择颜色**：打出万能牌后，在弹窗中选择颜色
4. **新游戏**：点击左上角「🔄 新游戏」按钮

### 游戏规则说明
- **可出的牌**：与当前颜色相同、与当前数字相同、同类型功能牌、万能牌
- **+2 叠加**：下家可以再出 +2 或 +4 继续叠加，否则抽全部叠加数量的牌
- **跳过**：下家被跳过，直接到下下家
- **反转**：出牌方向反转
- **万能牌**：可随时出，并指定下一种颜色
- **万能+4**：可随时出，下家被跳过并抽 4 张牌，同时指定颜色
